# Paper

This directory is the expected location of the source paper (`paper.pdf`).

The PDF is **not redistributed** here because the publisher (MDPI) is the
copyright holder. The article is open access (CC BY 4.0); download it from the
official DOI and save it as `paper/paper.pdf`:

- **Title:** A Dual-Segmentation Framework for the Automatic Detection and Size Estimation of Shrimp
- **Authors:** Malik Muhammad Waqar, Hassan Ali, Heng Zhou, Heba G. Mohamed, Sang Cheol Kim, Michal Strzelecki
- **Venue:** *Sensors* 2025, 25(18), 5830
- **DOI:** https://doi.org/10.3390/s25185830
- **PDF:** https://www.mdpi.com/1424-8220/25/18/5830/pdf

```bash
# From the repository root:
curl -L -o paper/paper.pdf "https://www.mdpi.com/1424-8220/25/18/5830/pdf"
```
